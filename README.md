# pycompoundv2
### Python wrapper for Compound v2 'Presidio'

Currently, config variables are retrieved from a 'config.ini' file. 

```
[ACCOUNT]
PrivateKey = *insert-private-key-here*
Address = *insert-address-here*

[PROVIDER]
InfuraMainnet = 
InfuraRinkeby = 
InfuraProjId = 
InfuraProjSecret = 
```
